## Como executar?

### Compilando e rodando
```
    git clone https://github.com/Wendlereis/teste-front-end.git
    cd teste-front-end
```
```
    npm install &&  npm start
    or
    yarn install && yarn start
```

Finalmente acesse:

 ```
    localhost:3000
```

### Executando testes

```
    npm test or yarn test
```
